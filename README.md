# AggressiveSlimes
## This game is like Angry Birds, where the player launches slimes at humans to take over the world. You drag back the slime and angle them to launch them into the buildings protecting the humans. Destroy all the humans to complete the level and be able to proceed to the next level.
## There are different slimes with abilities: the blue slimes are normal with no special powers, the red ones burn wooden structures, and the green ones melt metal structures. You have to strategically use these slimes to complete the level.
